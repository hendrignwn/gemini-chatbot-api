# Gemini Chatbot API

## Instalation

run code below:
```
npm install
```
after this, copy .env.example to .env, and fill variable ```GEMINI_API_KEY``` based on your api key in the gemini ai studio.

## Run

run server with code below:
```
node index.js
```

## Run in Browser
run chatbot in the browser with address url below:
```
http://localhost:3000
```
